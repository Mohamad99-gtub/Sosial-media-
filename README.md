# Belajar HTML, CSS, dan JavaScript

## Pendahuluan

Dalam belajar pengembangan web, ada tiga bahasa dasar yang harus dikuasai:

### 1. HTML (HyperText Markup Language)
HTML digunakan untuk membuat struktur dasar sebuah halaman web. Elemen-elemen umum yang digunakan di HTML meliputi:

- **`<a>`**: Elemen untuk membuat tautan.
  ```html
  <a href="https://www.example.com">Visit Example</a><button>: Elemen untuk membuat tombol.<button>Click Me</button><code>: Elemen untuk menampilkan kode program.<code>console.log('Hello World');</code><div>: Elemen untuk membungkus elemen lain.<div class="container">Content</div><p>: Elemen untuk paragraf teks.<p>This is a paragraph.</p><img>: Elemen untuk menampilkan gambar.<img src="image.jpg" alt="Description"><h1> - <h6>: Elemen untuk judul dengan tingkat kepentingan yang berbeda.<h1>Title</h1>
<h2>Sub Title</h2>2. CSS (Cascading Style Sheets)CSS digunakan untuk mengatur tampilan dan tata letak halaman web. Beberapa elemen dan properti yang umum digunakan meliputi:align-items: Properti untuk mengatur perataan item dalam kontainer flex..container {
  display: flex;
  align-items: center;
}background-color: Properti untuk mengatur warna latar belakang..element {
  background-color: #f00;
}color: Properti untuk mengatur warna teks..text {
  color: #000;
}display: Properti untuk mengatur cara elemen ditampilkan..block {
  display: block;
}font-size: Properti untuk mengatur ukuran font..text {
  font-size: 16px;
}margin: Properti untuk mengatur jarak luar elemen..element {
  margin: 10px;
}padding: Properti untuk mengatur jarak dalam elemen..element {
  padding: 10px;
}3. JavaScriptJavaScript adalah bahasa pemrograman yang digunakan untuk membuat halaman web lebih interaktif. Dengan JavaScript, kita bisa:Manipulasi DOM: Mengambil elemen HTML dan memodifikasinya.document.getElementById('example').innerHTML = 'Hello World';Menangani Peristiwa: Menangani peristiwa (event) seperti klik, hover, dll.document.getElementById('button').addEventListener('click', function() {
  alert('Button Clicked');
});Fungsi Animasi: Mendefinisikan animasi dengan keyframes.@keyframes example {
  from { background-color: red; }
  to { background-color: yellow; }
}
